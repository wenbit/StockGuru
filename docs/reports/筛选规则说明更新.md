## 📋 更新内容

### 新增内容

#### 1. **动量分数计算** 📈
详细说明了动量分数的计算公式和原理：

**计算公式**:
```
动量分数 = 斜率 × R² × 10000
```

**参数说明**:
- **斜率 (Slope)**: 线性回归计算过去25天价格趋势
- **R² (决定系数)**: 0-1之间，衡量趋势稳定性
- **放大系数**: 10000，便于比较

#### 2. **综合评分计算** ⚖️
详细的5步计算流程：

1. **获取数据**: 成交额Top100 ∩ 热度Top100
2. **Min-Max标准化**: `(值 - 最小值) / (最大值 - 最小值)`
3. **加权计算**: `0.5 × 成交额 + 0.5 × 热度`
4. **初选**: 取前30名
5. **终选**: 按动量排序，取前10名

#### 3. **筛选条件** 🔍
明确的筛选规则：

**排除条件** ❌:
- ST股票
- 次新股（上市<60天）

**必须满足** ✅:
- 成交额Top100
- 热度Top100
- K线数据充足（≥20天）

#### 4. **排序规则** 🏆
可视化的排名说明：

- 🥇 **金牌股票**: 第1-3名
- 🥈 **银牌股票**: 第4-10名

#### 5. **免责声明** ⚠️
强调风险提示和使用建议
